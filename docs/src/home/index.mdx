# Introduction

React ChatUI 는 ChatGPT나 Google Bard와 같은 챗봇형 AI의 대화를 표현하는 React 기반의 UI 라이브러리 입니다.
블로그나 웹사이트에 챗봇형 AI의 질문과 답변을 표시하려고 할 때 쉽게 적용하기 위해 만들어졌습니다.

## Features

- ⚛️ Ready for React v18;
- 🔥 Easy to embed question and answer of AI chatbot in your website.
- 🖥 Code block support;
- 📈 ChatGPT and Google Bard theme support;
- 📄 Custom themes;
- ⚡️ Offline Support & WebApp Manifest;
- and much more 🔥
